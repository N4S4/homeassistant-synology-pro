# Synology Pro

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-41BDF5.svg?style=for-the-badge)](https://github.com/hacs/integration)

Comprehensive Synology NAS integration for Home Assistant — sensors, binary sensors, switches, and services for every DSM subsystem.

[![Add to Home Assistant](https://my.home-assistant.io/badges/hacs_repository.svg)](https://my.home-assistant.io/redirect/hacs_repository/?owner=N4S4&repository=homeassistant-synology-pro&category=integration)

## What It Does

Monitor and control your Synology NAS directly from Home Assistant:

- 📊 **50+ sensors** — CPU, RAM, volumes, disks (SMART/temp), network, Docker
- ⚠️ **Binary sensors** — disk health, container status, security alerts  
- 🔘 **Switches** — Docker container start/stop
- 🤖 **5 services** — snapshots, container restart/update, security scans
- 🧩 **Config flow** — setup via Home Assistant UI (no YAML required)

## Quick Start

1. Click the **Add to Home Assistant** badge above
2. Add `https://github.com/N4S4/homeassistant-synology-pro` as custom repository
3. Install "Synology Pro" from HACS
4. Restart HA and add the integration via UI
